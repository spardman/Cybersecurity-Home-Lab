# Windows 11 Enterprise Endpoint
### Purpose: Target Hardening & GPO Policy Testing

## 🛠 Technical Specifications
* **OS:** Windows 11 Pro/Enterprise
* **Resources:** 2 vCPUs | 4GB RAM | 64GB VDI
* **Security Baseline:** Configured with TPM 2.0 and Secure Boot emulation.

## 🛡 Security Use Cases
* **Hardening:** Implementation of CIS Benchmarks for Windows 11.
* **Logging:** Configured Sysmon and Windows Event Forwarding for future SIEM integration.
* **Threat Simulation:** Used as a target for credential harvesting and lateral movement testing.
