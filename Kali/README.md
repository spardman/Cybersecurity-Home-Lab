# Kali Linux (Primary Attack Node)
### Purpose: Penetration Testing & Vulnerability Research

## 🛠 Technical Specifications
* **OS:** Kali Linux (Rolling Release)
* **Resources:** 2 vCPUs | 4GB RAM | 80GB VDI
* **Key Tools:** Nmap, Metasploit, Burp Suite, OpenVAS.

## 🛡 Security Use Cases
* **Vulnerability Scanning:** Running authenticated and unauthenticated scans against lab targets.
* **Network Discovery:** Mapping the internal NAT network to identify services and open ports.
* **GRC Reporting:** Translating raw technical scan data into business-risk assessments.
