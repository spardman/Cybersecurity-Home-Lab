# Ubuntu Server/Desktop 
### Purpose: Linux Infrastructure & Log Management

## 🛠 Technical Specifications
* **OS:** Ubuntu 22.04 LTS
* **Resources:** 1 vCPU | 2GB RAM | 25GB VDI
* **Role:** Web Server / Database Simulator

## 🛡 Security Use Cases
* **Access Control:** Auditing SSH configurations and implementing Key-Based Authentication.
* **Firewall Management:** Testing UFW (Uncomplicated Firewall) rules to block unauthorized traffic.
* **Log Analysis:** Monitoring `/var/log/auth.log` for brute-force attack attempts.
