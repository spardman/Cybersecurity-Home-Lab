# Parrot Security OS
### Purpose: Forensic Analysis & Secure Development

## 🛠 Technical Specifications
* **OS:** Parrot Security (MATE Edition)
* **Resources:** 2 vCPUs | 4GB RAM | 40GB VDI
* **Security Feature:** Sandbox mode enabled for secure tool testing.

## 🛡 Security Use Cases
* **Privacy Testing:** Utilizing built-in anonymity tools for threat intelligence gathering.
* **Forensics:** Testing memory dump analysis and file recovery tools in an isolated environment.
* **Scripting:** Writing and testing Python-based security scripts.
